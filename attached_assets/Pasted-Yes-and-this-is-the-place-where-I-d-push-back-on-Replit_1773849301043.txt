The key architectural point

You should not trend only derived scores.

You need three levels of trendability:

1. Raw trends

The original daily entries.

Example:
	•	novelty hunger rising
	•	sleep quality falling
	•	deep sleep rising
	•	noise aversion rising
	•	hormone duration rising

2. Composite trends

The calculated scores.

Example:
	•	ADHD pull
	•	Autism pull
	•	Recovery need

3. State inference trends

The interpreted system state.

Example:
	•	likely drifting toward Resensitization
	•	repeated Accumulation signatures
	•	Rebound window emerging

That third layer is the real goal.
